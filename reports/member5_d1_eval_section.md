# D1 Evaluation and Report Formatting

## Responsibilities
For Deliverable 1, my role focused on evaluation organization, report formatting, and documentation support for the Climate Evidence GraphRAG Agent project.

The responsibilities included:
- Organizing evaluation metrics for retrieval experiments
- Preparing a clean report structure
- Supporting metric comparison table formatting
- Assisting with section compilation from all team members
- Maintaining consistency in formatting and documentation style

---

## Evaluation Metrics
The following metrics were selected for evaluating retrieval quality and system performance:

- Recall@5
- NDCG@5
- Mean Reciprocal Rank (MRR)
- p95 Latency

These metrics help measure:
- retrieval relevance
- ranking quality
- answer support quality
- retrieval efficiency and latency
