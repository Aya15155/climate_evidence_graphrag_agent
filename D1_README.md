## AutoML Track A

The repaired AutoML experiment uses:

- **BM25 lexical retrieval**
- **TF-IDF + TruncatedSVD kNN retrieval**
- **Hybrid score fusion**
- **Optuna** with 30 trials

The search space is:

| Parameter | Values |
|---|---|
| `k` | 5, 10, 15, 20 |
| `metric` | cosine, euclidean, manhattan |
| `svd_dim` | 50, 100, 200, 300 |
| `normalization` | none, minmax, zscore |
| `hybrid_weight` | continuous float from 0.0 to 1.0 |

The objective is:

```text
NDCG@5 + 0.25 * Recall@5 - 0.0005 * p95_latency_ms
```

Current held-out test results:

| System | Recall@5 | NDCG@5 | MRR | p95 latency |
|---|---:|---:|---:|---:|
| Baseline | 0.667 | 0.522 | 0.493 | 674.5 ms |
| AutoML tuned | 0.833 | 0.682 | 0.645 | 676.2 ms |

Winning configuration:

```yaml
k: 20
metric: cosine
svd_dim: 300
normalization: minmax
hybrid_weight: 0.8152445172912974
```

Full reproducibility details are stored in:

```text
configs/run_card_d1.yaml
```

## Online learning

The River component performs **query-to-topic classification** across six climate topics:

- mitigation
- adaptation
- climate_science
- policy_governance
- technology_innovation
- uae_cop28

The simulated stream contains **600 queries** with concept drift injected at **query 350**. The learner uses **prequential evaluation** and ADWIN monitors the error stream.

Current results:

| Metric | Result |
|---|---:|
| First ADWIN alert | 377 |
| Pre-drift accuracy | 89.97% |
| Drift-window accuracy | 2.22% |
| Post-drift accuracy | 95.65% |
| Final cumulative accuracy | 78.33% |
| Final rolling accuracy | 100% |
